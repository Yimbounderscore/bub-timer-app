# Bub Timer App

Run app and view Bub yap time.

## How to Run
```powershell
npm install
npm start
```

## Build for Windows
```powershell
npm run build
```
Output: `dist/bub-timer-win32-x64/bub-timer.exe`

## Build for Linux (Fedora)
```powershell
npm run build:linux
```
Output: `dist/bub-timer-linux-x64/`
